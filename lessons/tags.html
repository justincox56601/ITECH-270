HTML Tags

The user of tags is the key characteristic of markup language. Markup language 'marks up' a document by surrounding parts of it with different tags. Those tags have an effect on how the content appears both to a human reader and to the computer. The internet users HTML. HTML stands for Hyper Text Markup Language. The hyper part of HTML refers to the ability to imnplement hyperlinks (links).

Tag Pairs

Most, but not all tags come in pairs. For instance the H1 tag used at the top of the page (HTML Tags) stands for Heading 1, and is written <H1>Your Cool Heading Here</H1>. Here are some of the most common tags you will use are listed below:

<H1></H1> - Heading 1, this is the primary heading of the page. Usually the page title.

<H2></H2> - Heading 2, Used as section headers.

<H3></H3> - Headings 3, 4, 5, 6, used as section headers of less importance than H2.

<p></p> - Pagagraph, used for writing text.

<a></a> - Anchor or hyperlinks, these allow you to make links or other pages or files.

<div></div> - Div, generic block level container.

<span></span> - Span, generic inline level container.
