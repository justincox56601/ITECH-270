Boilerplate

When creatintg a web page, we have some Boilerplate code that needs to be on every page. Below is an example of the Boilerplate code needed

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Document</title>
</head>
<body>
 Your content goes here.
</body>
</html>

<!DOCTYPE html>

This is a document type declaration. It tells the internet browser that this is an HTML document and needs to be interpreted as such.

<html lang="en">

This is the ROOT of the html document and contains ALL of the other tags.

<head>

This is the head tag. It sets the header information for the page such as css links, javascript links, document title, and meta information. This information is not meant to be viewable to the end user.

<meta>

This is a meta tag. It defines a piece of meta information about the page.

<body>

The body tags wrap all of the main readable content for the webpage.
