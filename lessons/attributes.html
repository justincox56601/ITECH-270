Attributes

Attributes are usually name/value pairs that add addtional information to html tags. All tags accept at least a few attributes, while some require them.

A very common attribute is the style attribute. style="color:red;" is an example of the style tag used for css.

Attributes are ALWAYS specified in the starting / opening tag <span style="color:red;">This text is colored red.</span>

anchor tags are an example of a tag that requires attritbutes to work correctly. <a href="https://google.com">Click here to go to Google</a> In the anchor tag we MUST supply the href attribute so the link can properly direct the user to the destination.

While most attributes are in the name/value format, not all of them are. For example the 'disabled' attribute for buttons is a stand alone attribute. The two buttons below are identical except for the 'disabled' attribute is added to the second one. Notice how you cannot click it.

<button>:Click Me</button>
<button disabled>:Click Me</button>
